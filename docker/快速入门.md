# 万宗心悟AI疗愈智能体 - 快速入门指南

## 系统要求

- Windows 10/11 + Docker Desktop
- 或 Linux + Docker + Docker Compose

## 一键启动（Windows）

```batch
# 进入docker目录
cd D:\hermes\项目\万宗心语\docker

# 双击运行 start.bat
start.bat
```

或者手动命令：

```batch
cd D:\hermes\项目\万宗心语\docker
docker-compose up -d
```

## 手动启动（Linux/Mac）

```bash
cd /path/to/万宗心语/docker

# 复制并编辑环境配置
cp .env.example .env  # 如有需要修改

# 启动所有服务
docker-compose up -d
```

## 启动后访问

| 服务 | 地址 |
|------|------|
| 用户端界面 | http://localhost |
| 管理后台 | http://localhost/admin |
| 后端API文档 | http://localhost:8000/docs |
| Qdrant向量库 | http://localhost:6333 |
| MinIO管理界面 | http://localhost:9001 |

## 初始化管理员账号

启动后需要创建管理员账号：

```bash
# 进入backend容器
docker exec -it wanzongxinyu_backend bash

# 创建管理员
python -c "from app.models.database import init_db, Admin; from app.services.auth import hash_password; init_db(); admin = Admin(username='admin', password_hash=hash_password('your_password'), is_admin=True); from app.models.database import db; db.add(admin); db.commit(); print('Admin created!')"
```

或者通过API：

```bash
curl -X POST http://localhost:8000/admin-api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"your_password"}'
```

## 查看服务状态

```bash
docker-compose ps
docker-compose logs -f backend  # 查看后端日志
```

## 停止服务

```bash
docker-compose down      # 停止并保留数据
docker-compose down -v   # 停止并删除数据（清空数据库）
```

## 常见问题

### 1. Docker未启动
```
[错误] Docker未启动，请先启动Docker Desktop
```
**解决**: 启动Docker Desktop，等待其完全运行后再试。

### 2. 端口冲突
```
Error: port is already allocated
```
**解决**: 检查占用端口的程序，或修改docker-compose.yml中的端口映射。

### 3. 前端无法访问
```
http://localhost 拒绝连接
```
**解决**:
1. 检查nginx是否运行: `docker-compose ps nginx`
2. 查看nginx日志: `docker-compose logs nginx`
3. 确认前端dist目录存在: `ls ../frontend/dist`

### 4. 数据库连接失败
等待15-30秒让数据库完全就绪后再试。

## 开发模式

如需修改代码后重新部署：

```bash
# 重新构建并部署
docker-compose down
docker-compose build --no-cache
docker-compose up -d
```

仅更新前端：
```bash
cd ../frontend
npm run build
docker-compose restart nginx
```
